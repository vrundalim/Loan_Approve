# Loan Approval Prediction System

## Project Overview
This project predicts whether a loan application will be approved or rejected based on applicant information using Machine Learning algorithms.

The system analyzes various applicant features and provides a prediction that can assist financial institutions in making loan approval decisions.

---

## Problem Statement
Loan approval is a critical process in the banking and financial sector. Manual evaluation can be time-consuming and prone to inconsistencies.

This project uses Machine Learning models to automate the prediction of loan approval status based on historical loan data.

---

## Dataset
The dataset contains applicant information such as:

- Income
- Loan Amount
- Credit History
- Education
- Employment Status
- Marital Status
- Dependents
- Other applicant-related features

Target Variable:
- Loan Approved (Yes/No)
